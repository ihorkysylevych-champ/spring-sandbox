# Spring Sandbox

### PDF Converter

Convert images to PDF on the fly.

```shell
GET http://localhost:8080/file/pdf/jpg
POST http://localhost:8080/file/convert/pdf (multipart upload)
GET http://localhost:8080/file/convert/{from}/{to}
```
